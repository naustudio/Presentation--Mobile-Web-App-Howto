What is a Mobile Web App?

An application

  • created with Web technology
  • on handheld form factors (smartphones, tablets)
  • run on mobile web browser* or WebView
  • normally having touch interface
  • offline or online connected through mobile network or wi-fi

* not include proxy browsers

Note on proxy browsers: browsers which rely on proxy servers to compress response data for bandwith and normally complex JS is stripped off.
